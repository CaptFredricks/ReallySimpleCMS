**Legend**
- N = new file
- D = deprecated file
- R = renamed file

- x.x.X (standard/minor release)
- x.x.x.X (bug fix/emergency patch release)
- x.x.x_snap-xx (snapshot release)

- [a] = alpha
- [b] = beta
